Teoría Teoría Ionic ¿Qué es ionic?

Framework SDK frontend de código abierto para desarrollar aplicaciones híbridas basado en tecnologías web HTML, CSS y JS.

Teoría Ionic Características Aplicaciones híbridas Buen diseño Emplea Capacitor HTML, CSS y Javascript Angular, React y Vue Conocimiento en desarrollo web Teoría Ionic Sesión práctica

Creación de una aplicación Ionic

$ npm install -g @ionic/cli

$ ionic start

$ cd myApp

$ ionic serve

$ ionic generate [options]
